USACO 2012 November Bronze - Find the Cow! - Solution [Java]

Explanation

This problem deals with lower numbers: the maximum length of the parenthetical string is 50,000. Thus, we can use nested loops as a complete search-esque type of problem. My solution is to assign each parenthesis, regardless of direction, in a character array. When a for loop (iterating through the char array) settles upon a "((", it will check all characters to the right for a case of "))".

By using this method, we do not overcount. Also, we account for every instance of hindlegs and forelegs that exist in the string.

Code

import java.io.*;
import java.util.*;

class cowfind{
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(new File("cowfind.in"));
        PrintWriter out = new PrintWriter(new File("cowfind.out"));

        String str = in.nextLine();
        char[] field = new char[str.length()+1];
        for(int i = 0; i < str.length(); i++){
            field[i] = str.charAt(i);
        }
        field[str.length()] = '0';
        int counter = 0;

        for(int i = 0; i < field.length; i++){
            if(field[i] == '(' && field[i+1] == '('){
                    int cursorStart = i+2;
                    for(int j = cursorStart; j < str.length()-1; j++){
                        if(field[j] == ')' && field[j+1] ==')'){
                            counter++;
                        }
                    }
            }
        }

        out.println(counter);
        out.close();
    }
}
