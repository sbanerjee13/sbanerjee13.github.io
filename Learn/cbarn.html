USACO 2016 February Bronze - Circular Barn - Solution [Java]

Explanation

The circular barn problem is simple in theory; the difficulty comes in the executable algorithm. As the input is single integers, an integer array is the optimal data type. Additionally, I have another integer variable that keeps track of the number of cows. While iterating through the input parameters, I am incrementing this variable. Thus, by the end of the input-taking method, my variable (called cowTotal) is set to the total number of cows.

Then comes the actual calculation. I set a minimum integer to Integer.Max_VALUE;. By doing so, my first iteration of the for loop algorithm would work. Within my for loop, I instance a new integer set to the total number of cows. Then, I run a nested for loop that adds the amount of cows staying in the same, singular room to a sum variable.

Code

< pre> import java.io.*; import java.util.*; class cbarn { public static void main(String[] args) throws IOException { Scanner in = new Scanner(new File("cbarn.in")); PrintWriter out = new PrintWriter(new File("cbarn.out")); int n = in.nextInt(); int[] cows = new int[n]; int cowTotal = 0; for(int i = 0; i < n; i++){ cows[i] = in.nextInt(); cowTotal += cows[i]; } int iterationCowTotal; int sum = 0; int minimum = Integer.MAX_VALUE; for(int i = 0; i < n; i++){ iterationCowTotal = cowTotal; sum = 0; for(int j = 0; j < n; j++) { iterationCowTotal = iterationCowTotal - cows[j]; sum += iterationCowTotal; } int temp=cows[n-1]; for(int k=n-1; k > 0;k--) { cows[k]=cows[k-1]; } cows[0]=temp; minimum = Math.min(minimum, sum); } out.println(minimum); out.close(); } }
