USACO 2013 December Bronze - Record Keeping - Solution [Java]

Explanation

In this problem, our input format is 3 strings (representing names of cows). These strings are in ALL CAPS. They are also not necessarily ordered. To summarize the algorithm, we need to collect the first letters of the names of the cows, order them alphabetically, and then record the number of times they enter the barn.

I decided to implement Java's LinkedHashMap and List objects in order to contain the data. The LinkedHashMap has keys in strings and values in integers. The string will be the ordered first letters of the 3 cows, and the integer will represent the number of times that group enters the barn.

Code

import java.util.*;
import java.io.*;

class records {

	public static void main(String[] args) throws IOException {

		Scanner in = new Scanner(new File("records.in"));
		PrintWriter out = new PrintWriter(new File("records.out"));

		int n = in.nextInt();
		in.nextLine();
		LinkedHashMap counter = new LinkedHashMap<>();
		List firstLetters = new LinkedList<>();
		for(int i = 0; i < n; i++){
			String cows = in.nextLine().replaceAll("\\s+", "");;
			System.out.println(cows);
			firstLetters.add(cows);
			if (counter.containsKey(cows)) {
				counter.put(cows, counter.get(cows)+1);
			} else {
				counter.put(cows, 1);
			}
		}

		int max = Integer.MIN_VALUE;
		for(int i = 0; i < n; i++){
			if(counter.get(firstLetters.get(i)) > max){
				max = counter.get(firstLetters.get(i));
			}
		}

		out.println(max);
		out.close();

	}
}
