Teleportation

USACO February 2018 Bronze

Here's an easy one! We simply determine whether the distance between Farmer John's initial location and the teleporter closest to him plus the distance between the second teleporter and the destination is smaller than if Farmer John just walks straight to the destination. It is important to note that the use of absolute value is helpful as it returns a distance that is always positive; by using positive numbers, we don't have to consider fixing the conditionals for walking backwards/forwards.

Problem Tags: Distance, Absolute Value, Quick Conditionals 

// @author Sid Banerjee, 2020         
            
import java.io.*;
import java.util.*;

public class teleport {
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(new File("teleport.in"));
        PrintWriter pw = new PrintWriter(new File("teleport.out"));

        // input here
        int a = in.nextInt(); int b = in.nextInt(); int x = in.nextInt(); int y = in.nextInt();

        // algorithm here
        int walkToDestination = Math.abs(b-a); // is this the optimal solution? walking from start to end?

        int closestToA; int closestToB;
        if(Math.abs(a-y) > Math.abs(a-x)){
            closestToA = x;
            closestToB = y;
        }else{
            closestToA = y;
            closestToB = x;
        }

        int distanceUsingTeleporters = Math.abs(a-closestToA) + Math.abs(b-closestToB);

        pw.println(distanceUsingTeleporters > walkToDestination ? walkToDestination : distanceUsingTeleporters);
        pw.close();
    }
}
