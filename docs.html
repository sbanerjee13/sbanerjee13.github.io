html quick docs

<html> - defines scope for html document [note: some browsers require the explicit instance of the !DOCTYPE html tag]

<div> - defines a container for elements to exist within

id [#], class [.] - defines scopes for tags in HTML for use in CSS. an ID is single-use; a class can define multiple elements

<head>, <body> - head defines pre-determined meta points, reference sheets, and external scripting files. body defines the content of the page.

<h[#]> - header content. can be changed via CSS.

<p> - paragraph content. flex and overflow{shown} by default. can be managed via CSS

<span> - defines inline elements

<img> - defines an image

<a href=""> - defines a link; href is the linkto {source}

<ul> - defines an unordered list.

<li> - defines a list item.

<meta> - defines a meta element. this definition is exemplified in the head section of any webpage.
