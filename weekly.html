Week 2: Virus!

Week of 2/24/2020 - 3/2/2020

Available languages: Java 11.0.5, Python 3.7.2, GNU G++17 7.3.0, C# Mono 5.18, Kotlin 1.3.10, PHP 7.2.13, ECMAScript 2019, Node.JS 12.16.1 LTS - you're welcome milan

You are the Mahavirus, a virulent and potent disease that can actually change the genetic code of humans, causing immunodeficiency (essentially, you are cancer). Given a string representing the genetic code of a human, determine the minimum amount of time it will take you to convert all of the genes into mahanine (M). However, you can only attack the gene in a specific way. Here are the rules:

1) You can only start by converting a guanine (G) into a mahanine (M). After the starting point, you can, in one unit of time, convert one more guanine and turn it into mahanine.
2) You can only infect a target to your left and/or right if they are adenine or guanine, in which case you can convert them into mahanine (M).
3) Any cytosine to your direct left and/or right you can convert into adenine.
4) The only way to convert a thymine into mahanine is to have it surrounded by mahanine.

For every unit of time, as many of the 4 "moves" as possible will occur. For example, if you have the string AGGGCCTCCGGAG, in one unit of time it will convert into AGGMCCTCCGGAG. Then, in time = 2, the string becomes AGMMACTCCMGAG. In time = 3, it will become AMMMMATCAMMAM. This continues until time = n, where the string is MMMMMMMMMMMMM.

INPUT:
The first and only line of the input is a string which represents the target human's genetic code line.

OUTPUT:
The output should be the minimum time it takes to fully convert the gene code into mahanine.

TEST INPUT:

AGGGCCTCCGGAG

TEST OUTPUT:

6

The answer of MMMMMMMMMMMMM can be created in 6 moves:

0) AGGGCCTCCGGAG
1) AGGMCCTCCGGAG
2) AGMMACTCCMGAG
3) AMMMMATCAMMAM
4) MMMMMMTCMMMMM
5) MMMMMMTMMMMMM
6) MMMMMMMMMMMMM

Note: This problem is not realistic. Though nucleic acid is encoded in G-C-A-T format, the lengths of the genetic codes used in this problem are not realistic lengths.
